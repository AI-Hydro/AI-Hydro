# Tool Documentation

Documentation of available Python tools and their capabilities.

## Files

- **tier1_libraries.json** - External library integrations (PySheds, Xarray, etc.)
- **tier2_wrappers.json** - AI-Hydro wrapper functions
- **tier3_workflows.json** - Complete workflow orchestrations
- **camels_tools.json** - CAMELS-specific tool documentation

## Purpose

Tool documentation helps the RAG engine:
- Recommend appropriate tools for user tasks
- Provide usage examples
- Explain tool capabilities and limitations
- Guide tool selection

## Tool Tiers

**Tier 1**: External libraries we integrate (PySheds, Rasterio, etc.)
**Tier 2**: Wrapper functions in ai_hydro package
**Tier 3**: Complete workflow functions
